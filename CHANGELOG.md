# Agent Eval Lite - Changelog

## [0.1.0-SNAPSHOT] - 2026-06-29

### Added
- Initial release
- Core evaluation framework with simple API
- Built-in scorers: Correctness, Safety, ResponseTime
- Support for custom scorers
- Concurrent evaluation with thread pool
- Comprehensive evaluation reports
- JUnit 5 test coverage

### Features
- Single entry point API: `Evaluator.evaluate()`
- Flexible test case format
- Multiple built-in metrics
- Easy custom scorer creation
- Detailed evaluation reports with pass/fail analysis
